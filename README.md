# CNN_Project_KSA
#### 교육과정: 2021 혁신성장인재 인공지능 과정
#### 장소: KSA 한국표준협회
#### 강사: 장우혁 강사, 우진철 박사(딥러닝개요 부분)
#### 교육내용: 딥러닝 기술 중 CNN 모델 이론 교육 및 실습

* CNN 개요
	- 딥러닝 개요
	- 합성곱(Convolution 연산)
	- 비용함수
	- 분류기(로지스틱, 소프트맥스)

	- 오버피팅
	- 가중치 규제

	- Dropout
	- BatchNomarliztion
	- 가중치 초기화
	- AlexNet 딥러닝 아키텍처
	- VGG-16, VGG-19 딥러닝 아키텍처
	- ResNet 딥러닝 아키텍처
	- DenseNet 딥러닝 아키텍처

* 실습
